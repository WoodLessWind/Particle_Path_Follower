# Particle Path Follower

一个用于 Unity 粒子系统（Particle System）的路径跟随组件。通过可视化的贝塞尔曲线（Bézier Curve）编辑器，让粒子能够沿着自定义的三维路径平滑运动，并支持丰富的位移散布与动态对齐控制。

## ✨ 核心特性

- **可视化路径编辑**：直接在 Scene 视图中操控贝塞尔曲线，支持拖拽、吸附和曲柄对称控制。
- **动态生命周期匹配**：粒子生成时可根据路径总长度和设定速度，自动计算并赋予合适的生命周期。
- **智能切线朝向**：粒子在运动过程中，其朝向和速度向量可自动对齐当前路径的切线方向。
- **高级位移控制**：
  - 支持多种偏移模式：**随机 (Random)**、**循环 (Repeat)**、**摇摆 (PingPong)**。
  - 支持非均匀散布范围界定（分别控制水平与垂直的随机上下限）。
  - 支持 **内部中空（Vacuum）** 机制，可轻松制作出空心管道等特效形状。
  - 支持 **强制圆形/椭圆（Circular Shape）** 约束计算。

## 🛠️ 使用指南

1. **挂载脚本**
   将 `ParticlePathFollower` 脚本挂载到带有 `Particle System` 的 GameObject 上（如果不带，脚本会自动挂载一个）。
2. **场景可视化编辑**
   - 选中该物体后，在 Scene 视图中可以看到绿色的节点（路径点）与深绿色的节点（手柄/控制点）。
   - **选中节点**：点击节点即可选中，随后拖拽坐标轴进行移动。
   - **吸附节点**：拖拽节点移动时**按住 `Shift` 键**，可以将其吸附到其他路径点上。
   - **恢复平滑手柄**：如果某个途径点的左右手柄失去对称导致曲线不平滑，可以选中其中一个手柄，并在**按住 `Shift` 键时点击**该手柄（或进行微小位移），即可将另一侧的手柄强制拉回对称状态。
3. **Inspector 工具操作**
   - **添加路径节点**：在 Inspector 面板下方点击“添加路径节点”，将在其末尾自动延伸出一段新路径。
   - **自动计算平滑**：点击“自动计算平滑”，脚本将通过自动推算，让所有转折路径点处的连接变得柔和、平滑。

## ⚙️ 参数详解 (Inspector)

### 运动设置

- **Speed**: 粒子沿路径移动的基准速度。
- **Auto Set Lifetime**: 勾选后，插件会根据路径总长度动态覆盖粒子系统的生命周期（`长度 / 速度`），确保粒子在其生命终点时正好抵达路径末端。
- **Align To Path**: 勾选后，粒子的渲染朝向（旋转角度）和实际受力方向（Velocity）会时刻紧贴当前的运动路径切线。

### 偏移设置

- **Offset Mode**: 决定粒子偏离主干路径的方式。
  - `None`: 不做任何偏移，严格沿中心线运动。
  - `Random`: 使粒子的出生拥有随机偏移，形成一条立体的轨迹带。
  - `Repeat`: 随生成时间呈重复或螺旋状排布偏移。
  - `PingPong`: 随时间呈现往复震荡波纹形偏移。
- **Offset**: 偏移范围。
  - `X / Y`: 代表 **水平方向（左右）** 偏移的 最小值 / 最大值。
  - `Z / W`: 代表 **垂直方向（上下）** 偏移的 最小值 / 最大值。
- **Enable Inner Vacuum (启用内部中空)**:
  - 勾选后可移除中心区域一定范围内的粒子散布，以此构建空心圆柱、管道特效。
  - **Inner Vacuum X / Y** 分别控制水平域和垂直域掏空区域的百分比边界（0.0 ~ 1.0）。
- **Circular Shape**: 默认偏移基于矩形范围，勾选此项强制将其约束为圆形或椭圆散布。
- **Offset Frequency**: 在 `Repeat` 和 `PingPong` 模式下生效，控制偏移循环的节奏频率。
